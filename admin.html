Local Tool

Episode Builder

Google Docsから貼り付けた原稿を、NeuroRadioのエピソード追加用JSONに整えます。

Paste from Google Docs

以下のラベルを使うと自動で読み取れます: Episode, Title, Date, Spotify, Topics, Summary, Show Notes, Editorial Notes.

Generated Draft

保存したJSONを `tools/add-episode.mjs` に渡すと、個別ページとトップページが自動更新されます。

Then run

node tools/add-episode.mjs /tmp/neuroradio-new-episode.json
